# **lms-frontend**

# 软件项目管理前端

## 环境构建及调试
- 最好使用VSCode打开，安装Vetur插件
- 下载NPM，并执行：
```
npm install
```
- 启动本地网页调试（自动同步修改）
```
npm run serve
```
- 单元测试
```
npm run test:unit
```
- 构建可部署包
```bash
npm run build
```
